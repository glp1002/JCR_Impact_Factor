{% trans %}Modelos de predicción{% endtrans %}

{% trans %}Para predecir el valor del Índice de Impacto de las revistas, nuestra aplicación utiliza tres modelos de predicción: AdaBoost, Random Forest y XGB Regressor.{% endtrans %} {% trans %}Estos modelos son técnicas de aprendizaje automático que se aplican en el campo de la ciencia de datos para realizar predicciones basadas en datos históricos y características relevantes de las revistas.{% endtrans %}

  • {% trans %}El AdaBoost es un algoritmo que combina múltiples modelos de aprendizaje débiles para crear un modelo más robusto y preciso. Cada modelo débil se entrena secuencialmente, poniendo más énfasis en las instancias mal clasificadas en cada iteración. El resultado final es un modelo de predicción que mejora su capacidad de clasificación a medida que se agregan más modelos débiles.{% endtrans %}
  • {% trans %}El Random Forest es un algoritmo que crea un "bosque" de árboles de decisión individuales. Cada árbol se entrena de forma independiente en una muestra aleatoria de las características y ejemplos de entrenamiento. Luego, para realizar una predicción, cada árbol emite una decisión y el resultado final se determina por votación o promedio de las predicciones de todos los árboles.{% endtrans %}
  • {% trans %}El XGB Regressor, también conocido como Gradient Boosting, es un algoritmo de refuerzo que construye un modelo de regresión a partir de múltiples modelos de árbol de decisión. En cada iteración, el modelo se entrena para corregir los errores residuales del modelo anterior, lo que permite mejorar progresivamente la precisión de las predicciones.{% endtrans %}
{% trans %}En cuanto a los valores de RMSE (Root Mean Square Error) obtenidos por cada modelo, se utiliza como métrica para evaluar la proximidad a las predicciones del Índice de Impacto real de las revistas. El RMSE mide la diferencia promedio entre las predicciones y los valores reales, indicando un valor más bajo una mayor precisión del modelo. En nuestro caso, los valores de RMSE para cada modelo son los siguientes: {% endtrans %}
  • AdaBoost: 5.56
  • Random Forest: 4.85
  • XGB Regressor: 4.58

{% trans %}Guía de inicio rápida{% endtrans %}

{% trans %}En la interfaz principal podrá visualizar dos menús desplegables donde seleccionar la categoría y la revista que interese. Es preciso seleccionar primero la categoría, que hace un filtro inicial de las revistas.{% endtrans %}

{% trans %}Llegado a este punto, se activará el botón "Histórico JCR", que le redireccionará a una nueva interfaz. Podrá visualizar los siguientes elementos:{% endtrans %}

  • {% trans %}Gráfica con la evolución del valor del Índice de Impacto de la revista durante los últimos 5 años disponibles en el JCR de Clarivate.{% endtrans %}
  • {% trans %}Gráfico de barras con el cuartil al que ha pertenecido la revista durante los 5 últimos años disponibles en el JCR de Clarivate.{% endtrans %}

{% trans %}Volviendo a la interfaz principal, bajo los menús desplegables, encontrará los distintos modelos disponibles para predecir el Índice de Impacto de la revista seleccionada. Se puede seleccionar más de uno simultáneamente.{% endtrans %}

{% trans %}Tras seleccionar los modelos deseados, se activará el botón "Predecir JCR" que redireccionará a una nueva interfaz. Podrá visualizar los siguientes elementos:{% endtrans %}

  • {% trans %}Aclaración textual con el valor de la predicción para el siguiente año (respecto al último disponible en el JCR de Clarivate).{% endtrans %}
  • {% trans %}Gráfico con la evolución temporal del Índice de Impacto de la revista en los 5 últimos años disponibles en el JCR de Clarivate. Adicionalmente, se pueden visualizar las predicciones de cada modelo y su leyenda.{% endtrans %}
